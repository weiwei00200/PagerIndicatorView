# PagerIndicatorView
**介绍：该控件可用于制作Banner或App首次进入的引导页。**
 
### 使用方法
##### 步骤1：
implementation 'com.github.weiwei00200:PagerIndicatorView:1.0.0'
    
##### 步骤2：
allprojects {
   repositories {
      maven { url "https://jitpack.io" }
   }
}
